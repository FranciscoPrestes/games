Match de Letras

A parte mais bá para a construção de uma Expressão Regular (RegEx) são os caracteres simples. As RegEx mais básicas são compostas apenas por caracteres simples, como a, 1 e C.

Exemplo:
  • A RegEx z reconhece o padrão z
  • A RegEx abcde reconhece o padrão abcde
  • A RegEx q não reconhece o padrão m

A exceção à essa regra são alguns caracteres, que possuem um significado especial dentro das RegEx's ($, ^, *, (, ), +, {, }, [, ], \, |, ?, -). Para utilizá-los como caracteres normais, você precisa colocar uma \ antes deles.

Exemplo:
  • A Regex $ não reconhece o padrão $
  • A Regex \$ reconhece o padrão $
  • A Regex ( não reconhece o padrão ( e é inválida.
